# var和:=

|          | `var`  | `:=`              |
| -------- | ------ | ----------------- |
| **作用范围** | 全局+函数内 | 函数内               |
| **赋值**   | 可不赋值   | 必须赋值              |
| **类型声明** | 可以显式声明 | 不能显式声明            |
| **重新声明** | 不可重新声明 | 左侧有至少一个新变量时，可重新声明 |

> **何时使用`var`**
> - 声明全局变量
> - 声明但不赋值
> - 显式指定类型
