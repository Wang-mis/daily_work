## LeetCode（5 题）

### 2463. 最小移动总距离（04.14）
- **解法：动态规划**
- 机器人和工厂分布在 X 轴上，将工厂按位置展开（重复 limit 次），对机器人和工厂排序后，用 DP 计算分配机器人到工厂的最小总移动距离。
- 关键思路：排序后最优分配一定是有序的（不会交叉），将问题转化为区间 DP。

### 2515. 到目标字符串的最短距离（04.15）
- **解法：线性遍历**
- 给定环形字符串数组，从起始下标出发，找到目标字符串的最短顺/逆时针距离。
- 关键思路：正向遍历找最近的目标位置，逆向遍历找另一方向，取二者最小值。

### 3488. 距离最小相等元素查询（04.16）
- **解法：哈希表 + 二分查找**
- 环形数组中，对每个查询 `(i, k)`，找距离 `nums[i]` 最近的值等于 `nums[k]` 的元素。
- 关键思路：用哈希表记录每个值出现的所有下标，对每次查询二分查找最近位置，同时考虑环形距离。

### 3761. 镜像对之间最小绝对距离（04.17）
- **解法：排序 / 哈希表**
- 找数组中所有"镜像对"（`nums[i] + nums[j] == 0`）之间下标差的最小绝对值。
- 关键思路：用哈希表记录每个值的下标列表，遍历时对每个负数查找其正数镜像，计算下标差最小值。

### 3783. 整数的镜像距离（04.18）
- **解法：模拟**
- 计算 `abs(n - reverse(n))`，其中 `reverse(n)` 为将整数各位反转后的数值。
- 关键思路：直接字符串反转后转整数，做差取绝对值即可。

### 9915. Highest Cost Orders（04.17）
- **平台**：StrataScratch
- **知识点**：WITH 子句（CTE）、窗口函数
- 用 CTE 计算每个客户的最高订单金额，结合窗口函数排名筛选结果。
- **重点**：`WITH` 子句用于将复杂子查询命名复用，窗口函数 `ROW_NUMBER() / RANK() OVER(PARTITION BY ... ORDER BY ...)` 用于分组排序。

### 10318. New Products（04.18）
- **平台**：StrataScratch
- **知识点**：CASE-WHEN、COUNT vs SUM 区别
- 统计每个公司在 2020 年新增（相比 2019 年）的产品数量。
- **重点**：`CASE-WHEN` 用于条件分支；`COUNT` 统计非 NULL 行数，`SUM` 对值求和——`SUM(CASE WHEN ... THEN 1 ELSE 0 END)` 与 `COUNT(CASE WHEN ... THEN 1 END)` 效果等价但语义不同。

| 知识点 | 要点 |
|--------|------|
| HTTPS（04.14） | 在 HTTP 基础上加 TLS/SSL 加密层；握手过程：非对称加密协商对称密钥，后续通信用对称加密；证书由 CA 签发，防止中间人攻击 |
